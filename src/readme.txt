You can embed your WordPress posts in other websites in a similar way you embed social media. Post Embeds plugin enhance the default core post embed functionality by adding customizations and design options.

== Description ==

Post Embeds plugin makes easy to customize your WordPress post embeds. You can make your post embeds use a similar design to Twitter and Facebook with just a few of clicks.

Customizing posts embeds has never been easier! Want to display the author but not the date and time? [Post Embeds](https://postembeds.htmgarcia.com/) makes this process very simple.

## Main features

- Support for post and pages
- Apply the customizations by post type
- 3 design styles available. More to come!
- Display options for date and time
- Display options for read more link
- Display options for author
